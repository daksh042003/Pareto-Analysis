## Overview

<p style="font-size:16px;">
This project demonstrates an advanced Power BI Pareto Analysis dashboard built using
<strong>Visual Calculations</strong>, <strong>Field Parameters</strong>, and dynamic interactivity,
following modern Power BI best practices.
</p>

---

## Project Overview

<p style="font-size:16px;">
The dashboard enables business users to analyze <strong>Sales or Profit concentration</strong>
across <strong>Products, Customers, or Countries</strong>, applying the Pareto Principle
(80/20 rule) with a dynamic threshold selector.
</p>

<p style="font-size:16px;">
All Pareto logic is implemented using <strong>Power BI Visual Calculations</strong>, ensuring
flexibility, performance, and compliance with the latest calculation layer.
</p>

---

## Key Features

- Dynamic Pareto Threshold (50%–95%) with real-time reclassification of Top Contributors  
- Metric switching between Sales and Profit  
- Dimension switching using Field Parameters:
  - Product
  - Customer
  - Country
- Pareto Combo Chart with:
  - Descending sort by contribution
  - Cumulative % line
  - Dynamic threshold reference line
  - Conditional bar coloring based on Pareto category
- KPI summary cards for executive-level insights  
- Detailed Pareto table with Rank, Cumulative Total, Individual %, and Cumulative %  
- Professional UI/UX aligned with OpenBI branding  

---

## Technical Highlights

- All Pareto calculations (Rank, Running Total, Cumulative %, Individual %, Category)
  implemented using <strong>Visual Calculations</strong>
- Minimal traditional DAX used only for base aggregations
- Fully interactive single-page report (1920 × 1080)
- Embedded dataset (no external connections)
